## Overview
This repository contains a complete sentiment analysis project on Amazon product reviews. Leveraging advanced NLP techniques and machine learning, it processes over 194,000 reviews to classify sentiments (Positive, Neutral, Negative), generate visualizations, and build predictive models. Ideal for e-commerce businesses, marketers, and data scientists to extract actionable insights from customer feedback.

**Key Highlights:**
- **Dataset**: Amazon reviews with fields like reviewText, overall rating, and timestamps.
- **Core Workflow**: Data preprocessing ‚Üí Sentiment scoring ‚Üí EDA ‚Üí Visualizations ‚Üí ML modeling.
- **Advanced Elements**: VADER sentiment analysis, TF-IDF vectorization, Naive Bayes classification, and interactive Plotly charts.

---

##  Features
- **Data Preprocessing**: Robust text cleaning with NLTK (tokenization, lemmatization, stopword removal).
- **Sentiment Scoring**: VADER for accurate polarity and subjectivity analysis.
- **Exploratory Analysis**: Word clouds, correlation plots, and trend analysis over time.
- **Visualizations**: Interactive charts (pie, bar, scatter, line) using Plotly and static plots with Matplotlib/Seaborn.
- **Machine Learning**: Custom sentiment classifier with Scikit-learn (Naive Bayes), including evaluation metrics.
- **Scalability**: Handles large datasets with sampling and error handling.
- **Extensibility**: Supports multilingual reviews and API integrations.

-------

## Technologies Used
- **Programming Language**: Python 3.8+
- **Core Libraries**:
  - Data Handling: Pandas, NumPy
  - NLP: NLTK, VADER Sentiment
  - ML: Scikit-learn (TF-IDF, Naive Bayes)
  - Visualization: Matplotlib, Seaborn, Plotly, WordCloud
- **Tools**: Jupyter Notebook, GitHub for version control.

-----

### Example Output
- **Sentiment Distribution**: Pie chart showing ~65% Positive, 25% Negative, 10% Neutral.
- **Word Cloud**: Visual highlights of key terms in positive/negative reviews.
- **Model Accuracy**: ~82% on test data with detailed classification reports.

-----

##  Results & Insights
- **Sentiment Breakdown**: Majority of reviews are positive, correlating with higher ratings.
- **Key Trends**: Negative sentiments often link to product durability; positive to quality and value.
- **ML Performance**: Naive Bayes model achieves high accuracy, enabling automated classification.
- **Business Impact**: Insights can inform product development, customer service, and targeted marketing.
